## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool
- **React Router** - Navigation
- **Lucide React** - Icons
- **date-fns** - Date formatting
- **CSS Variables** - Theming

## Usage

1. **Home Screen** - View upcoming events or create a new one
2. **Create Event** - Fill in details, upload a background image, and save
3. **Event Detail** - View event info, RSVP, and share with guests
4. **Invite Modal** - Copy link, send email, or share via messaging apps
