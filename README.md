# Invites App

A beautiful event invitation app replica inspired by Apple's Invites app. Create events, invite guests via email or message, and manage RSVPs.

## Features

- **Create Events** - Add event title, date/time, location, and custom background images
- **Invite Guests** - Send invitations via email, SMS, or WhatsApp
- **RSVP Tracking** - Guests can respond with Going, Not Going, or Maybe
- **Shared Albums** - Upload and share photos from the event
- **Shared Playlists** - Coming soon!
- **Beautiful Dark UI** - Glass morphism design with smooth animations

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool
- **React Router** - Navigation
- **Lucide React** - Icons
- **date-fns** - Date formatting
- **CSS Variables** - Theming

## Usage

1. **Home Screen** - View upcoming events or create a new one
2. **Create Event** - Fill in details, upload a background image, and save
3. **Event Detail** - View event info, RSVP, and share with guests
4. **Invite Modal** - Copy link, send email, or share via messaging apps
